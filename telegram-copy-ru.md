# Telegram copy (RU)

## Entry point button

`Импорт из существующего OpenClaw`

## Intro message

Если у вас уже есть свой OpenClaw, можно перенести его в CloudClaw почти без ручной настройки.

Мы создадим одноразовую защищённую ссылку для загрузки архива вашего текущего инстанса. Дальше вы просто вставите готовую инструкцию в свой старый OpenClaw.

## Pre-flight warning

Что будет перенесено:
- память
- настройки ассистента
- файлы workspace
- ключи, токены и другие секреты — только после явного подтверждения в старом инстансе

Важно:
- импорт работает только в пустой новый инстанс
- архив хранится временно
- максимальный размер архива: 5 ГБ

## Prompt delivery message

Откройте свой старый OpenClaw и отправьте ему это сообщение целиком:

`<вставить prompt из public-agent-prompt.md>`

Когда старый OpenClaw ответит `UPLOAD_OK`, вернитесь сюда и нажмите кнопку ниже.

## Finish button

`Завершить импорт`

## Import running

Импортирую данные в новый инстанс. Это может занять несколько минут.

## Success

Готово. Новый инстанс поднят на основе вашего предыдущего OpenClaw.

## Failure: archive too large

Архив получился больше 5 ГБ. Сначала нужно уменьшить его размер: удалить временные файлы, кэш или тяжёлые ненужные данные.

## Failure: expired upload

Ссылка для загрузки устарела. Нажмите импорт ещё раз — я создам новую.
